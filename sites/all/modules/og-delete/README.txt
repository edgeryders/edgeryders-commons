------------------------------------------------------------------------------
  OG Delete Module for Drupal
------------------------------------------------------------------------------

Description
===========

This module will provides the given below option to user while deleting the OG Node.

Delete all the content associated with group.

Delete the group and make the content public.

If you chose move all group posts above, specify a destination group.

Limitation
==========

Currently this module works for node content type basis.

Dependencies
============

In order to delete OG Node via the user interface, OG Delete depends on:

* OG
